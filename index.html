Back-end

Desenvolvimento de software responsável por realizar as tarefas invisíveis ao usuário final, como processamento de dados, integração com bancos de dados e comunicação com as aplicações do front-end.

Front-end

Desenvolvimento de software responsável por tudo o que o usuário final vê e interage em uma aplicação web ou móvel, incluindo a interface gráfica, animações, interações com o usuário e apresentação de dados.

Mobile

Desenvolvimento de aplicativos utilizando a linguagem java e kotlin para dispositivos móveis, como smartphones. Isso inclui a concepção, programação e teste de aplicativos que sejam otimizados para serem executados.

Soft Skills
